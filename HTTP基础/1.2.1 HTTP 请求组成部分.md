# **HTTP 请求** 的三个核心组成部分


## 1️ HTTP 请求结构概览

一个完整的 HTTP 请求由三部分组成：

```
请求行（Request Line）
请求头（Request Headers）
请求体（Request Body）
```

> 注：请求体在 **GET 请求通常为空**，POST/PUT/PATCH 等请求可能会包含请求体。

---

## 2️ 请求行（Request Line）

**请求行**是 HTTP 请求的第一行，包含三个部分：

```
方法 请求路径 HTTP版本
```

* **方法（Method）**：客户端想对资源执行的操作。

  * 常见方法：`GET`, `POST`, `PUT`, `DELETE`, `PATCH`, `HEAD`, `OPTIONS`
* **请求路径（URL Path）**：资源的路径，例如 `/index.html`。
* **HTTP 版本**：客户端使用的 HTTP 协议版本，如 `HTTP/1.1`、`HTTP/2`。

**示例**：

```
GET /index.html HTTP/1.1
```

> 含义：客户端请求服务器的 `/index.html` 文件，使用 HTTP/1.1 协议。

---

## 3️ 请求头（Request Headers）

**请求头**是一些 **键值对信息**，用于描述客户端、请求属性或告诉服务器如何处理请求。

* 作用：

  * 描述请求的客户端环境（浏览器、操作系统）
  * 告诉服务器数据类型、压缩方式、语言偏好等
  * 携带身份认证信息（Cookie、Authorization）

**常用请求头示例**：

| 头部字段           | 说明                             |
| -------------- | ------------------------------ |
| Host           | 指定请求的主机名和端口（HTTP/1.1 必须）       |
| User-Agent     | 客户端信息，如浏览器类型                   |
| Accept         | 客户端可接受的响应类型（MIME 类型）           |
| Content-Type   | 请求体的数据类型，例如 `application/json` |
| Content-Length | 请求体的字节长度                       |
| Cookie         | 浏览器保存的 Cookie 信息               |

**示例**：

```
Host: www.example.com
User-Agent: Mozilla/5.0
Accept: text/html,application/xhtml+xml
```

---

## 4️ 请求体（Request Body）

**请求体**是可选部分，主要用于 **POST、PUT、PATCH** 等方法发送数据给服务器。

* **作用**：

  * 传递表单数据、JSON 数据、文件等。
* **特点**：

  * GET/DELETE 请求一般没有请求体。
  * 请求体格式由 `Content-Type` 指定。

**常见请求体示例**：

* **表单数据**（`application/x-www-form-urlencoded`）：

```
username=alice&password=123456
```

* **JSON 数据**（`application/json`）：

```json
{
  "username": "alice",
  "password": "123456"
}
```

---

## 5️ 一个完整 HTTP POST 请求示例

```
POST /login HTTP/1.1
Host: www.example.com
User-Agent: Mozilla/5.0
Content-Type: application/json
Content-Length: 44

{
  "username": "alice",
  "password": "123456"
}
```

* 第一行是 **请求行**
* 中间部分是 **请求头**
* 最后一部分是 **请求体**

---

总结：

| 部分  | 内容           | 作用               |
| --- | ------------ | ---------------- |
| 请求行 | 方法 + 路径 + 版本 | 指明请求目标和方式        |
| 请求头 | 键值对          | 描述请求属性、客户端信息、认证等 |
| 请求体 | 请求内容         | 发送数据给服务器（可选）     |
